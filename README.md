# 📚 Book Scraper Bot

## 🧠 Description
This project is a Python automation script that uses Selenium to scrape book names and prices from a website and saves the data into a CSV file.

## ⚙️ Technologies Used
- Python
- Selenium
- Pandas

## 🚀 Features
- Automatically opens browser
- Extracts book names and prices
- Saves data into CSV file

## ▶️ How to Run
1. Install required libraries:
   pip install selenium pandas webdriver-manager

2. Run the script:
   python scraper.py

## 📂 Output
- my_books_data.csv
